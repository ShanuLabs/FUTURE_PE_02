## 🌐 Project Overview

This project was developed as part of the **Future Interns Prompt Engineering Internship – Task 2**.

The objective of this project is to create a complete **AI-powered User Generated Content (UGC) Marketing Campaign** for a fictional premium specialty coffee café using advanced Prompt Engineering techniques.

The project focuses on generating engaging, authentic, and conversion-focused marketing content for multiple social media platforms.

## 🎯 Project Objective

To design an AI-powered UGC marketing campaign that helps a local café:

- Increase Brand Awareness
- Improve Customer Engagement
- Boost Social Media Reach
- Generate More Café Visits
- Increase Online Orders
- Improve Customer Trust
- Build a Strong Digital Presence
